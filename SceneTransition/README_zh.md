# 沐曦-自动驾驶场景转换解决方案

[英文版](README.md) | 中文版

## 介绍
该解决方案面向自动驾驶研究领域。它利用 MetaX C500 GPU 强大的训练和推理能力，结合用于场景转换的视频编辑技术，可以方便的实现数据场景迁移，例如将晴朗天气的街道转变为大雾天气。
